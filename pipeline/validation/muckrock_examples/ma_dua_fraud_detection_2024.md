# Benefits fraud detection system studies

## Request scope (verbatim from MuckRock page)

> Pursuant to the Massachusetts Public Records Law, I hereby request records
> related to ID verification and unemployment fraud detection tools used by
> the department.
>
> I request: 1) All contracts the department has entered into since Jan. 1,
> 2020 for the provision of unemployment fraud detection tools. This
> includes identity verification tools and data analytics services such as
> risk-scoring and fraud-scoring algorithms.

## Characterization

A direct ADS-scope MA public-records request filed against the Department
of Unemployment Assistance in January 2024, partially completed. The
request explicitly asks for contracts of identity-verification and
fraud-detection / risk-scoring / fraud-scoring algorithms — i.e., the
exact record class V2's `procurement_contracts_rfps` module targets — and
also requests accuracy and disparate-flagging analyses, mapping to V2's
`validation_accuracy_audits` module.
